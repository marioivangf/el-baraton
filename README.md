# El baratón

Para correr el proyecto es necesario tener instalado Node.js (>=8.0) recomendado tener instalado [yarn](https://yarnpkg.com/es-ES/docs/install), luego sólo hay que correr (desde el root del proyecto): 

```bash
yarn && yarn start
```
Si lo deseas también puedes usar npm `npm i && npm run start`

Con ello se inicializará el proyecto en modo de desarrollo en el puerto [1234](http://localhost:1234), con livereload activado en automático

## Dist

Para crear un build de disitribución `yarn build`, los archivos generados se instalarán en `/dist`
